Barq-WASM browser benchmark harness

Methodology: SIMD support is feature-detected before loading a bundle; every workload's result is verified against the scalar reference before timing; warm-up runs precede measurement; statistics are median/p90/p95/stddev over many samples with performance.now(). The JavaScript baseline runs the same algorithm with the same warm-up and sample counts on the same data. Environment metadata is recorded with the results. No numbers are published anywhere from this page automatically — use the JSON download and record the hardware.
